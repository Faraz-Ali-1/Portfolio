About

As a web developer, my focus is on turning ideas beautifully into the web. I spent considerable time completing the comprehensive curriculum from freeCodeCamp, building a strong hand in HTML, CSS, and JavaScript, often speeding up my design process with Tailwind CSS.

Outside of writing code, I treat collaboration and organization as key skills. I rely on Git and GitHub for version control and use the Command Line to manage my development environment.

My goal isn't just to write code, but to solve real-world problems. I'm actively working towards expanding my backend knowledge so I can take on larger, end-to-end development roles and contribute to innovative projects that make a difference.

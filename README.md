# Portfolio

Personal Portfolio Website

This is my professional portfolio website, designed to showcase my journey, technical skills, and the projects I've built. It is fully responsive.


Built With

* **HTML5** – Semantic structure for better SEO and accessibility.
* **CSS3** – Custom styling using **Flexbox** and **CSS Grid** for complex layouts.
* **JavaScript** – For interactive elements and smooth animations.

## ✨ Features

* **Responsive Layout:** Optimized for mobile, tablet, and desktop screens.
* **Project Gallery:** A curated list of my best work with links to source code and live demos.
* **Contact Integration:** Easy way for recruiters and collaborators to reach out.
* **Optimized Performance:** Clean code and compressed assets for fast loading.

### Technical Highlights

* **Custom Animation Engine:** Built using the **Intersection Observer API** to trigger staggered reveals and state changes without the overhead of heavy libraries.
* **SVG Motion Graphics:** Designed a multi-orbit SVG hero scene with synchronized CSS keyframe rotations to create visual depth and a modern tech aesthetic.
* **Dynamic UI States:** The navigation bar and global theme intelligently shift colors (e.g., to Modern Green) based on the user's scroll position via JavaScript.
* **Performance First:** Optimized for Core Web Vitals using lazy-loading for assets and efficient CSS variable-based theming.
